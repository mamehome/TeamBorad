## Drive同期

`teamboard-drive-apps-script.gs` をGoogle Apps Scriptに貼り付けて、ウェブアプリとしてデプロイしてください。
発行URLとチームキーをTeamBoard右上の「同期」に設定します。

## 注意

ZIPを解凍後、`index.html` だけを取り出さず、同じフォルダ内に `styles.css` と `app.js` がある状態で開いてください。

## v21 追加：資料ページ

- 下部タブに「資料」を追加
- 座学やミーティングで使った資料を保存可能
- PDF / PowerPoint（.pdf / .ppt / .pptx）対応
- 資料名、対象カテゴリー、日付、メモを保存
- 資料一覧からPDFを開く / PowerPointを保存
- Drive同期とJSON保存に資料データも含まれます

注意：
PDFやPowerPointはBase64として端末内・Drive同期データに含めます。
ファイルサイズが大きいとブラウザ保存や同期が重くなるため、12MB以下を目安にしてください。
