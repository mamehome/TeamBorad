## Drive同期

`teamboard-drive-apps-script.gs` をGoogle Apps Scriptに貼り付けて、ウェブアプリとしてデプロイしてください。
発行URLとチームキーをTeamBoard右上の「同期」に設定します。

## 注意

ZIPを解凍後、`index.html` だけを取り出さず、同じフォルダ内に `styles.css` と `app.js` がある状態で開いてください。

## v6 追加

- TOP画面にチームロゴ表示エリアを追加

## v7 追加

- 配置パーツに「T テキスト」を追加
- テキストを選択してピッチをタップすると、入力した文字を図に配置できます
- 配置したテキストは保存・プレビュー・Drive同期の対象になります

## v8 追加

- 配置パーツに「ゴール」を追加
- 配置パーツに「ペナルティエリア」を追加
- パーツの向きを 45° 単位で左回転・右回転できる操作を追加
- 新規作成時は、線や矢印などが置かれていない空のピッチから開始します
- ゴール、ペナルティエリア、テキストなども保存・プレビュー・Drive同期の対象です
